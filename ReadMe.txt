Principe de fonctionnement du code et données utilisées
 
Le téléservice CREIC permet à tout demandeur communautaire inactif d'effectuer des demandes en ligne et de déposer les justificatifs correspondants en vue de son rattachement au régime d'assurance maladie français.
Le téléservice ne réalise pas en lui-même des traitements métier : l'instruction des demandes est réalisée par les agents de l'Assurance Maladie.
Pour des raisons de sécurité, ce code source a été expurgé, avant publication, des parties sensibles concernant les mécanismes d’authentification et d’habilitation, ainsi que de la liste des bibliothèques tierces embarquées.
Le document "Description du code TLS-CREIC.pdf" explique le fonctionnement du code publié.

Code Functionality and Data Used
  
The CREIC online service allows any nonworking Europeean national to submit applications online and provide the required supporting documents in order to be affiliated with the French health insurance system.
The online service does not itself carry out any business processing: applications are reviewed and handled by National Health Insurance staff.
For security reasons, this source code has been purged, prior to publication, of sensitive parts concerning authentication and authorization mechanisms, as well as the list of third-party embedded libraries.
The document “Description du code TLS-CREIC.pdf” explains how the published code works.
